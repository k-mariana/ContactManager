# ContactManager
An ASP.NET MVC application that allows users to upload a CSV file with the following fields, store the data into MS SQL database, and display stored data on the page. 
### CSV fields:
- Name [string]
- Date of birth [date]
- Married [bool]
- Phone [string]
- Salary [decimal]
